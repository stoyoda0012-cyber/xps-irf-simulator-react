XPS IRF シミュレータ

コンセプト & 物理モデル

概要

本シミュレータは、X線光電子分光法（XPS）システムの装置応答関数（IRF: Instrumental Response Function）をモデル化します。 IRFは、理想的なスペクトルが測定システムによってどのようにブロードニング・歪みを受けるかを記述します。

座標系

シミュレータは2D座標系を使用します：

  • X軸（エネルギー）：フェルミ準位からの束縛エネルギー（meV）
  • Y軸（空間）：スリット方向の位置（mm）

物理モデル

1. フェルミ・ディラック分布

金属のフェルミ端における理想的なスペクトルは以下に従います：

f(E, T) = 1 / (1 + exp(E / kT))

ここで k = 8.617 × 10⁻⁵ eV/K（ボルツマン定数）

2. 合成分解能モデル

総IRFは光源と検出器の寄与の畳み込みです：

IRFtotal = IRFsource ⊗ IRFdetector

ガウシアンブロードニングの場合、合成分解能は：

σcombined = √(σsource² + σdetector²)

3. X線源パラメータ

パラメータ 記号 説明
光源分解能 σsrc X線源の固有エネルギー分解能（meV）
スポット幅 σY 試料上のX線スポットの空間的広がり（mm）
エネルギー歪度 γE 光源エネルギー分布の非対称性
空間歪度 γY スポット空間分布の非対称性
エネルギー勾配 α スポット内のエネルギー変化（dE/dy）

4. 2D検出器パラメータ

パラメータ 記号 説明
スマイル曲率 κ アナライザー収差による放物線歪み
検出器傾き θ 検出器取り付け角度のズレ
検出器分解能 σdet 2D検出器の固有分解能（meV）

シミュレーションパイプライン

  1. フェルミ・ディラック分布：指定温度での理想的なステップ関数を生成
  2. 2D発光：X線スポットの空間分布を通してスペクトルを投影
  3. 検出器歪み：スマイル曲率と傾きの変換を適用
  4. 分解能畳み込み：光源と検出器のガウシアンカーネルで畳み込み
  5. 1D投影：空間軸に沿って積算し、観測スペクトルを取得
  6. ノイズ付加：ポアソンおよびガウシアンノイズを適用

重要な洞察

本シミュレータは、幾何学的歪み（スマイル、傾き）と分解能成分（光源、検出器）が どのように組み合わさって観測されるIRF非対称性を生み出すかを示します。 パラメータをインタラクティブに調整することで、各成分が測定スペクトルに どのような影響を与えるかを理解できます。
